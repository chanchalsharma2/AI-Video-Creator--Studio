# 🎬 AI Video Creator Studio
A high-speed content automation engine powered by **Llama-3.3** (via Groq API) and **Streamlit**.

## 🚀 Overview
This project is a professional-grade tool designed for content creators. It automates the process of generating viral video titles, full scripts, SEO-friendly descriptions, and trending keywords using Large Language Models (LLMs).

## ✨ Key Features
- **⚡ Lightning Fast:** Powered by Groq's Llama-3.3-70b model for near-instant responses.
- **🛠️ Structured Output:** Delivers data in a clean, professional format (JSON-ready logic).
- **📝 Complete Package:** Generates titles, hooks, body content, and SEO metadata in one click.
- **📥 Downloadable:** Users can download the entire script as a `.txt` file for offline use.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Frontend:** Streamlit (Modern Dashboard UI)
- **AI Engine:** Groq Cloud API (Llama-3.3-70b-versatile)
- **Environment:** Dotenv for secure API management

## 📦 Installation & Setup
1. Clone the repository:
   ```bash
   git clone [https://github.com/chanchalsharma2/AI-Video-Creator--Studio.git](https://github.com/chanchalsharma2/AI-Video-Creator--Studio.git)

 * Install dependencies:
   pip install -r requirements.txt

 * Set up your .env file with your GROQ_API_KEY.
 * Run the application:
   streamlit run app.py
